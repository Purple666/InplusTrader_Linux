InplusTrader 从数据获取、数据分析、算法交易、回测引擎、实盘模拟、实盘交易，为程序化交易者提供了全套解决方案。

InplusTrader 具有灵活的配置方式，强大的扩展性，用户可以非常容易地定制专属于自己的程序化交易系统。

InplusTrader 结合最新人工智能、机器学习等技术，助力量化投资。
